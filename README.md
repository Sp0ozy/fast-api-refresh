# FastAPI Refresh

A Todo app built while revisiting [FastAPI - The Complete Course](https://www.udemy.com/course/fastapi-the-complete-course/) on Udemy.

## What I revised and practiced

- **FastAPI basics** - routing, path/query parameters, request bodies, response models, status codes
- **Pydantic** - data validation with `BaseModel` and `Field`
- **SQLAlchemy** - defining models, sessions, CRUD operations with a SQLite database
- **Authentication** - JWT tokens, password hashing with bcrypt, OAuth2 password flow
- **Routers** - splitting the app into modules (`auth`, `todos`, `admin`, `user`)
- **Alembic** - database migrations
- **Unit testing** - writing tests with `pytest`, using fixtures and mocking dependencies
- **Templating** - serving HTML pages with Jinja2 and connecting them to the API via fetch (JavaScript)
